🏡 Airbnb Data Analysis Dashboard
📊 Overview

This project showcases an interactive Airbnb Dashboard that analyzes listing data to uncover insights about pricing, availability, location trends, and property distribution.

The dashboard helps users understand how Airbnb listings vary across different regions and what factors influence pricing and popularity.

📁 Dataset
File: AirBnb_tutorial_Data.xlsx
Includes:
Listing ID
Host information
Location (neighborhood, latitude, longitude)
Price
Availability
Room type
Number of reviews
